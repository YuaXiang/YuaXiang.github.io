快速排序、堆排序、归并排序

发表于 2017-08-23 | 分类于 算法 |

1.快速排序

快速排序主要是基于分治的思想，每次寻找一个pivot，然后从后往前找比pivot小的，从前往后找比pivot小的。

2.堆排序

堆排序主要分三个步骤

堆调整，比较当前元素及其左右孩子，把最大的调整上来（大顶堆）,递归调整。

建堆，根据堆的性质，从nums.length/2-1 最后一个非叶子结点开始从后往前建堆。

排序，从后往前交换堆顶元素和堆最后一个元素，减少堆元素，进行堆的调整。

3.归并排序
